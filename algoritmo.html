4 Como o Array.sort() compara duas linhas sort(a, b)

Para cada par de linhas, o algoritmo consulta o mapa e compara os indices. Quem tem indice menor vem antes.

Linha A (id 001)
prioridade = "baixa"
priorityMap.get("baixa") = 3
3 − 0 = 3 > 0
B vem antes
Linha B (id 002)
prioridade = "critica"
priorityMap.get("critica") = 0
Linha A (id 005)
prioridade = "alta"
priorityMap.get("alta") = 1
1 − 2 = −1 < 0
A vem antes
Linha B (id 003)
prioridade = "media"
priorityMap.get("media") = 2
// Funcao de comparacao do sort
sortedData = [...rawData].sort((a, b) => {
  const pa = priorityMap.get(a[coluna]) ?? Infinity;
  const pb = priorityMap.get(b[coluna]) ?? Infinity;
  return pa - pb;  // negativo = a antes, positivo = b antes
});
